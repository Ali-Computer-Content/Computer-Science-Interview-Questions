1::What is a computer network?::A collection of interconnected devices that can communicate and share resources.::Networks enable data sharing, communication, and resource utilization like printers and servers.
2::What are the main goals of computer networks?::Resource sharing, reliability, scalability, and communication.::These goals improve efficiency and availability of systems.
3::What is the OSI model?::A 7-layer reference model for network communication.::It standardizes communication functions for interoperability.
4::How many layers are in the OSI model?::Seven.::Physical, Data Link, Network, Transport, Session, Presentation, Application.
5::Which OSI layer handles bit transmission?::Physical layer.::It deals with voltage, cables, and signaling.
6::Which layer is responsible for framing?::Data Link layer.::Frames encapsulate packets for node-to-node delivery.
7::Which OSI layer provides logical addressing?::Network layer.::IP addresses are used here.
8::Which layer ensures reliable delivery?::Transport layer.::Protocols like TCP operate here.
9::Which OSI layer manages sessions?::Session layer.::It establishes, manages, and terminates sessions.
10::Which layer handles encryption and compression?::Presentation layer.::It formats data for applications.
11::Which layer interacts with users?::Application layer.::Protocols like HTTP and FTP reside here.
12::What is the TCP/IP model?::A 4-layer networking model used on the Internet.::It simplifies OSI into practical layers.
13::Name TCP/IP layers.::Application, Transport, Internet, Network Access.::Each maps to OSI layers.
14::What is a protocol?::A set of rules for communication.::Ensures correct data exchange.
15::What is IP?::Internet Protocol.::Provides logical addressing and routing.
16::What is IPv4 address size?::32 bits.::Written as four octets.
17::What is IPv6 address size?::128 bits.::Designed to overcome IPv4 exhaustion.
18::What is a MAC address?::A 48-bit hardware address.::Used at Data Link layer.
19::What is ARP?::Address Resolution Protocol.::Maps IP to MAC addresses.
20::What is RARP?::Reverse Address Resolution Protocol.::Maps MAC to IP.
21::What is TCP?::Transmission Control Protocol.::Provides reliable, connection-oriented service.
22::What is UDP?::User Datagram Protocol.::Provides fast, connectionless service.
23::Difference between TCP and UDP?::TCP is reliable, UDP is not.::TCP ensures delivery, UDP favors speed.
24::What is a port number?::Logical endpoint for communication.::Allows multiple services on one IP.
25::Well-known port for HTTP?::80.::Used for web traffic.
26::Well-known port for HTTPS?::443.::Encrypted web communication.
27::What is FTP?::File Transfer Protocol.::Used for file transfer.
28::FTP port number?::21.::Control connection.
29::What is DNS?::Domain Name System.::Translates domain names to IP addresses.
30::DNS port?::53.::Uses UDP primarily.
31::What is DHCP?::Dynamic Host Configuration Protocol.::Automatically assigns IP addresses.
32::DHCP port numbers?::67 and 68.::Client-server communication.
33::What is ICMP?::Internet Control Message Protocol.::Used for error reporting.
34::What is ping?::ICMP echo request tool.::Tests connectivity.
35::What is traceroute?::Path discovery tool.::Shows hops to destination.
36::What is routing?::Selecting paths for packet forwarding.::Handled by routers.
37::What is a router?::A Layer 3 device.::Forwards packets using IP addresses.
38::What is a switch?::A Layer 2 device.::Uses MAC addresses to forward frames.
39::What is a hub?::A Layer 1 device.::Broadcasts data to all ports.
40::Difference between hub and switch?::Switch is intelligent, hub is not.::Switch reduces collisions.
41::What is collision domain?::Network segment where collisions occur.::Switch reduces collision domains.
42::What is broadcast domain?::Area receiving broadcast frames.::Routers break broadcast domains.
43::What is subnetting?::Dividing a network into smaller networks.::Improves efficiency and security.
44::What is subnet mask?::Defines network and host bits.::Used with IP address.
45::Default subnet mask for Class C?::255.255.255.0.::24 network bits.
46::What is CIDR?::Classless Inter-Domain Routing.::Allows flexible subnetting.
47::What is NAT?::Network Address Translation.::Maps private IPs to public IP.
48::Types of NAT?::Static, Dynamic, PAT.::Used for IP conservation.
49::What is PAT?::Port Address Translation.::Multiple devices share one public IP.
50::What is firewall?::Network security device.::Controls incoming and outgoing traffic.
51::What is IDS?::Intrusion Detection System.::Detects malicious activity.
52::What is IPS?::Intrusion Prevention System.::Detects and blocks threats.
53::What is VPN?::Virtual Private Network.::Creates secure tunnel over public network.
54::What is SSL/TLS?::Security protocols.::Encrypt data in transit.
55::What is HTTPS?::HTTP over TLS.::Secure web communication.
56::What is packet switching?::Data sent in packets.::Efficient and flexible.
57::What is circuit switching?::Dedicated communication path.::Used in traditional telephony.
58::What is bandwidth?::Maximum data rate.::Measured in bps.
59::What is latency?::Transmission delay.::Lower latency improves performance.
60::What is throughput?::Actual data transfer rate.::Depends on network conditions.
61::What is jitter?::Variation in delay.::Affects real-time traffic.
62::What is QoS?::Quality of Service.::Prioritizes network traffic.
63::What is congestion?::Network overload.::Causes packet loss.
64::What is flow control?::Regulating data transmission.::Prevents receiver overflow.
65::What is error control?::Detecting and correcting errors.::Ensures data integrity.
66::What is checksum?::Error detection mechanism.::Used in TCP/UDP.
67::What is CRC?::Cyclic Redundancy Check.::Detects transmission errors.
68::What is sliding window?::Flow control mechanism.::Used by TCP.
69::What is three-way handshake?::TCP connection establishment.::SYN, SYN-ACK, ACK.
70::What is TCP teardown?::Connection termination.::FIN and ACK exchanges.
71::What is MTU?::Maximum Transmission Unit.::Largest packet size.
72::What is fragmentation?::Breaking packets into smaller pieces.::Occurs when MTU exceeded.
73::What is reassembly?::Recombining fragments.::Done at destination.
74::What is proxy server?::Intermediate server.::Improves security and performance.
75::What is load balancer?::Traffic distribution device.::Improves availability.
76::What is VLAN?::Virtual LAN.::Logical network segmentation.
77::VLAN operates at which layer?::Data Link layer.::Uses tagging.
78::What is STP?::Spanning Tree Protocol.::Prevents switching loops.
79::What is ARQ?::Automatic Repeat reQuest.::Error control method.
80::What is stop-and-wait ARQ?::Send one frame at a time.::Simple but inefficient.
81::What is go-back-N ARQ?::Resend multiple frames.::Improves efficiency.
82::What is selective repeat ARQ?::Resend only lost frames.::Most efficient.
83::What is unicast?::One-to-one communication.::Single receiver.
84::What is multicast?::One-to-many communication.::Group-based delivery.
85::What is broadcast?::One-to-all communication.::Sent to all hosts.
86::What is CSMA/CD?::Ethernet access method.::Detects collisions.
87::What is CSMA/CA?::Wireless access method.::Avoids collisions.
88::What is Wi-Fi standard?::IEEE 802.11.::Wireless LAN specification.
89::What is Bluetooth?::Short-range wireless technology.::Uses IEEE 802.15.
90::What is LTE?::Long Term Evolution.::4G cellular technology.
91::What is 5G?::Next-generation cellular network.::High speed and low latency.
92::What is MAN?::Metropolitan Area Network.::Covers a city.
93::What is LAN?::Local Area Network.::Small geographic area.
94::What is WAN?::Wide Area Network.::Large geographic area.
95::What is PAN?::Personal Area Network.::Very small range.
96::What is ISP?::Internet Service Provider.::Provides Internet access.
97::What is backbone network?::High-speed core network.::Connects major nodes.
98::What is BGP?::Border Gateway Protocol.::Inter-domain routing.
99::What is RIP?::Routing Information Protocol.::Distance vector protocol.
100::What is OSPF?::Open Shortest Path First.::Link-state routing protocol.
101::What is metric in routing?::Cost value.::Determines best path.
102::What is hop count?::Number of routers crossed.::Used by RIP.
103::What is packet loss?::Dropped packets.::Occurs due to congestion.
104::What is sniffing?::Packet capturing attack.::Threatens confidentiality.
105::What is spoofing?::Impersonation attack.::Fake identity usage.
106::What is DoS attack?::Denial of Service.::Overwhelms network resources.
107::What is DDoS?::Distributed DoS.::Attack from multiple sources.
108::What is man-in-the-middle attack?::Intercepting communication.::Compromises confidentiality.
109::What is encryption?::Data transformation for security.::Protects confidentiality.
110::What is symmetric encryption?::Same key for encrypt/decrypt.::Fast but key sharing issue.
111::What is asymmetric encryption?::Public/private key pair.::Used in key exchange.
112::What is RSA used for?::Key exchange and signatures.::Asymmetric algorithm.
113::What is hashing?::One-way function.::Used for integrity.
114::What is digital signature?::Cryptographic authentication.::Ensures integrity and authenticity.
115::What is PKI?::Public Key Infrastructure.::Manages certificates.
116::What is X.509?::Certificate standard.::Used in SSL/TLS.
117::What is session key?::Temporary symmetric key.::Used per session.
118::What is replay attack?::Reusing captured packets.::Threatens authentication.
119::What is firewall rule?::Traffic filtering rule.::Controls access.
120::What is stateful firewall?::Tracks connection state.::More secure.
121::What is stateless firewall?::Filters based on rules only.::Faster but less secure.
122::What is proxy firewall?::Application-level filtering.::Deep inspection.

124::What is network topology?::Network layout.::Physical or logical.

129::What is full duplex?::Simultaneous send/receive.::Improves throughput.
130::What is half duplex?::One direction at a time.::Used in older systems.
131::What is simplex?::One-way communication.::Like broadcast TV.
132::What is encapsulation?::Adding headers.::Each layer adds info.
133::What is decapsulation?::Removing headers.::At receiving end.
134::What is socket?::IP + port combination.::Identifies connection endpoint.
135::What is congestion control?::Managing traffic load.::TCP implements it.
136::What is slow start?::TCP congestion control phase.::Gradual rate increase.
137::What is window size?::Amount of data sent without ACK.::Controls flow.
138::What is ACK?::Acknowledgement.::Confirms receipt.

140::What is TTL?::Time To Live.::Limits packet lifespan.
