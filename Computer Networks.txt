1::What is a computer network?::A computer network is a set of interconnected devices that can communicate and share resources with each other.
2::What are the types of networks?::LAN (Local Area Network), WAN (Wide Area Network), MAN (Metropolitan Area Network), PAN (Personal Area Network).
3::What is the OSI model?::The OSI model is a conceptual framework with 7 layers: Physical, Data Link, Network, Transport, Session, Presentation, and Application.
4::What are the functions of the Physical layer?::The Physical layer transmits raw bits over a physical medium and defines voltage, timing, and data rates.
5::What is the Data Link layer responsible for?::The Data Link layer handles error detection, flow control, and frames data for transmission between adjacent nodes.
6::What is the Network layer responsible for?::The Network layer manages logical addressing, routing, and forwarding of packets across networks.
7::What is the Transport layer responsible for?::The Transport layer provides reliable or unreliable data delivery, segmentation, and flow control.
8::What is the Session layer responsible for?::The Session layer manages sessions or connections between applications on different devices.
9::What is the Presentation layer responsible for?::The Presentation layer handles data translation, encryption, compression, and formatting.
10::What is the Application layer responsible for?::The Application layer provides network services directly to user applications, like HTTP, FTP, SMTP.
11::What is TCP/IP model?::TCP/IP is a protocol suite with 4 layers: Link, Internet, Transport, and Application.
12::What is the difference between OSI and TCP/IP models?::OSI has 7 layers and is conceptual; TCP/IP has 4 layers and is protocol-based and widely used.
13::What is a protocol?::A protocol is a set of rules that define how data is transmitted and received over a network.
14::What is the difference between connection-oriented and connectionless communication?::Connection-oriented (TCP) establishes a connection before data transfer; Connectionless (UDP) sends data without establishing a connection.
15::What is TCP?::TCP (Transmission Control Protocol) is a connection-oriented, reliable transport protocol that ensures data delivery.
16::What is UDP?::UDP (User Datagram Protocol) is a connectionless, lightweight transport protocol with no guarantee of delivery.
17::What is the difference between TCP and UDP?::TCP is reliable, ordered, and connection-oriented; UDP is unreliable, unordered, and connectionless.
18::What is IP address?::An IP address is a unique identifier for a device on a network, either IPv4 (32-bit) or IPv6 (128-bit).
19::What is the difference between IPv4 and IPv6?::IPv4 uses 32-bit addresses; IPv6 uses 128-bit addresses and supports a larger address space.
20::What is subnetting?::Subnetting divides a network into smaller sub-networks to improve routing efficiency and security.
21::What is a MAC address?::A MAC (Media Access Control) address is a unique hardware identifier assigned to a network interface.
22::What is ARP?::ARP (Address Resolution Protocol) maps an IP address to a MAC address in a local network.
23::What is RARP?::RARP (Reverse ARP) maps a MAC address to an IP address.
24::What is DHCP?::DHCP (Dynamic Host Configuration Protocol) automatically assigns IP addresses and other network configuration to devices.
25::What is DNS?::DNS (Domain Name System) translates domain names into IP addresses.
26::What is the difference between static and dynamic IP addressing?::Static IP is manually assigned and fixed; Dynamic IP is automatically assigned by DHCP and can change over time.
27::What is NAT?::NAT (Network Address Translation) allows multiple devices on a private network to share a single public IP address.
28::What is PAT?::PAT (Port Address Translation) allows multiple devices to share a single IP using different port numbers.
29::What is a default gateway?::A default gateway is a router that forwards traffic from a local network to other networks.
30::What is a switch?::A switch is a network device that connects devices in a LAN and forwards data based on MAC addresses.
31::What is a hub?::A hub is a network device that broadcasts data to all devices in a network segment.
32::What is the difference between a hub and a switch?::Hub broadcasts to all ports; switch forwards to specific ports using MAC addresses.
33::What is a router?::A router connects different networks and forwards packets based on IP addresses.
34::What is the difference between a router and a switch?::Router operates at Layer 3 (Network) and forwards packets between networks; Switch operates at Layer 2 (Data Link) and forwards frames within a LAN.
35::What is a VLAN?::A VLAN (Virtual LAN) is a logical segmentation of a network to separate broadcast domains within a single physical network.
